# HOW TO INSTALL YOLO IN VITIS-AI

1. Follow [link](https://github.com/Xilinx/Vitis-AI) to install Vitis-AI.

2. 
``` shell
git clone https://github.com/ultralytics/yolov5.git
```

3. Follow this [link](https://www.hackster.io/LogicTronix/yolov5-quantization-compilation-with-vitis-ai-3-0-for-kria-7b005d) to chaneg the sigmoid function.

4. Copy insp.py to the root directory of yolov5.

5. run the script by **python3 insp.py**
